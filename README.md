## Getting Started
To run Dating App on your local machine, follow these steps:

1. **Clone the Repository**
    ```bash
    git clone https://github.com/mobilions/Dating-ProviderXPattern
    cd Dating-ProviderXPattern
    ```

2. **Install Dependencies**
    ```bash
    flutter pub get
    ```

3. **Run the App**
    ```bash
    flutter run
    ```

## Requirements
